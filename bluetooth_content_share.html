St Wills Academy Website

A responsive educational website built to showcase programs, admission details, and contact information for St Wills Academy.

Features

- Fully responsive (mobile and desktop)
- Modern hero section with call-to-action
- Programs displayed using card layout
- Contact page with direct call and WhatsApp integration
- Clean and structured UI for better user experience

Tech Stack

- HTML
- CSS (Mobile-first approach)
- Basic JavaScript

Project Structure

- index.html (Homepage)
- pages/contact.html
- pages/programs.html
- css/style.css
- images/
- video/

Status

Frontend structure completed. Ready for feedback and further improvements.
